# Религиовед. Реконструкция утраченной культуры

Этот репозиторий используется как вспомогательная база для настольной кооперативной исследовательской игры, посвящённой реконструкции религиозной системы исчезнувшей культуры эпохи Зелёной Сахары.

Во время партии игроки получают доступ к отдельным страницам этого сайта через QR-коды на карточках артефактов.

Репозиторий не предназначен для последовательного чтения или самостоятельного изучения вне игрового процесса.

---

## Как используется этот сайт

- В начале партии игроки не имеют доступа ко всем материалам.
- По мере исследования артефактов они переходят по QR-кодам и читают соответствующие страницы.
- Раздел `basics/` используется как справочный и методологический.
- Раздел `artifacts/` содержит подробные описания находок.
- Раздел `images/` содержит иллюстрации и визуальные материалы.

---

## Структура репозитория

- `basics/` — базовые тексты и концептуальные материалы  
- `artifacts/` — каталог артефактов (открывается по ходу игры)  
- `images/` — изображения и иллюстрации
